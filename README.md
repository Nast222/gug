# GitHub User Finder

## Автор: [артемьева]

## Описание
Приложение для поиска пользователей GitHub с возможностью добавления в избранное и сохранения списка в JSON.

## Как использовать API
- Введите логин пользователя в поле поиска.
- Нажмите «Поиск» для отображения информации.
- Кнопка «В избранное» добавляет пользователя в список избранных.
- Избранные сохраняются в файл `favorites.json`.

## Примеры использования
1. Введите `torvalds` и нажмите «Поиск» — отобразится информация о Линусе Торвальдсе.
2. Добавьте пользователя в избранное.
3. Нажмите «Показать избранное» — откроется окно со списком.
4. Перезапустите приложение — избранные сохранятся.
